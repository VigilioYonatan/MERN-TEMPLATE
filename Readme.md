# MERN TEMPLATE - VIGILIO

1. Instalacion Express & React Vite

```bash
# Express
cd server
yarn install

# React Vite
cd client
yarn install
```

2.  Variables de entorno

-   Crear archivo `.env` y completar igualmente de .example.env

3. Ejecutar Sevidor EXPRESS && react vite

```bash
# Express
yarn start
yarn dev
```

```bash
# React vite
yarn dev
```
